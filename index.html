MacPatch

An open source management suite for large Mac OS X environments.

MacPatch on GitHub

Tweet Follow

Something

Users can patch at their own convenience, or simply wait to be patched after a scheduled time.


Other thing

User can easily see the number of outstanding patches, and check their patch state at a glance.

Patch Management

Delpoy Apple and third party patch content.

Inventory

Collect inventory data on your entire Mac fleet.

Software Catalog

Users can self install software from your list of approved packages.

Scalable

Deploy as many secondary servers as needed for the size or your fleet.

Configurable

Settings such as scaning and patching can be configured.

Open Source

The entire code is avaliable on GitHub.

Software Catalog

Users can install software from a catalog that you manage. No admin required. Create muliple catalogs for different groups of users.

Patch Groups

You can create multiple patch groups. Each patch group can be configured with a different patch baseline. For example create one patch group for your servers and one for desktops.

Terminal

Most features can be accessed from the command line for remote administrating or scripting.
